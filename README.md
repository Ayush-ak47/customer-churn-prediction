# 📉 Customer Churn Prediction

## 📌 Problem Statement
Customer churn is a major challenge for businesses. The objective of this project is to build a machine learning model that predicts whether a customer is likely to leave (churn) or stay with the company. This helps businesses take proactive actions to retain customers.

---

## 📊 Dataset
- Dataset contains customer information such as:
  - Demographics (gender, senior citizen, etc.)
  - Account details (tenure, contract type, billing)
  - Services used (internet, phone, etc.)
  - Target variable: **Churn (Yes/No)**  

---

## 🛠️ Tech Stack
- **Programming:** Python  
- **Libraries:**  
  - Pandas, NumPy  
  - Matplotlib, Seaborn  
  - Scikit-learn  

---

## ⚙️ Project Workflow

### 1. Data Cleaning
- Handled missing values  
- Converted categorical variables into numerical format  

### 2. Exploratory Data Analysis (EDA)
- Analyzed customer behavior patterns  
- Visualized churn distribution  
- Identified important features affecting churn  

### 3. Feature Engineering
- Encoded categorical variables  
- Scaled numerical features  

### 4. Model Building
- Applied machine learning algorithms such as:
  - Logistic Regression  
  - Decision Tree / Random Forest  

### 5. Model Evaluation
- Evaluated using:
  - Accuracy  
  - Confusion Matrix  
  - Classification Report
